# Cotta

Cotta is a personal AI assistant with clear boundaries and traceable behavior.

## Philosophy

- **Local first**: the service and interface run in your own environment, and you decide when it starts and stops.
- **Minimum necessary capability**: it only presents capabilities that are actually implemented, without assuming broader permissions by default.
- **Honest about boundaries**: it states clearly what it can and cannot do, without pretending to be more capable than it is.
- **Traceable by design**: sessions, messages, run states, and logs can be reviewed so you can understand what happened.

Cotta is not trying to be all-powerful. It is trying to be clear, restrained, and trustworthy.
